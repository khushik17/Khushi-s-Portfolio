# 🍄 Khushi Kabra — Mario Portfolio

> A Super Mario themed interactive portfolio website built with pure HTML, CSS and JavaScript.
---

## 🎮 How to Play

| Key | Action |
|-----|--------|
| `←` `→` or `A` `D` | Move Mario left / right |
| `ENTER` | Open panel when near a pipe or animal |
| `Click` | Click coins, pipes, animals directly |

---

## 🌿 What's Inside

| Element | Content |
|---------|---------|

| 🐾 Animals | Skills (Frontend, Backend, Databases, Languages, Tools) |
| 🪙 Coins | Collect them while walking! |
